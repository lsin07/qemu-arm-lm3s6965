# QEMU ARM LM3S6965 Bare Metal Example with VSCode Cortex-Debug

## Prerequisites

### Required

- ARM GUN Toolchain
- Make
- QEMU

### Optional

- Visual Studio Code (for automated build and debug)
- [Cortex-Debug Extension](https://marketplace.visualstudio.com/items?itemName=marus25.cortex-debug) from Visual Studio Code

## How to Run

**Using VS Code**: press `F5` to run VSCode Debugger.

**Without VS Code**: type `make run` on a terminal
